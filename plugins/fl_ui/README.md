# fl_ui

Reusable UI widgets for this Flutter base template — buttons, inputs, badges, dropdowns, dialogs, and layout helpers not tied to a specific feature.

Import via `package:core/core.dart` (re-exported) or directly via `package:fl_ui/fl_ui.dart`.

See [`.agents/skills/fl-ui-components/SKILL.md`](../../.agents/skills/fl-ui-components/SKILL.md) for a catalog of what's here, and [`lib/fl_ui_previews.dart`](lib/fl_ui_previews.dart) / `flutter widget-preview start` for a live preview.
