# fl_media

Media handling plugin for this Flutter base template — image/video viewing (`ImageView`, `MediaViewerWidget`), video playback control (`VideoControllerManager`), image cropping, and native file picking (`PickFileHelper`), plus their localization.

Import via `package:core/core.dart` (re-exported) or directly via `package:fl_media/fl_media.dart`.

See [`.agents/skills/fl-ui-components/SKILL.md`](../../.agents/skills/fl-ui-components/SKILL.md) for a catalog of what's here, and [`example/`](example/) for a runnable demo.
