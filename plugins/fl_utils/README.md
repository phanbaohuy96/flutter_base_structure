# fl_utils

Framework-agnostic helpers for this Flutter base template: extensions, date/number/phone formatting, JWT decoding, encryption, debouncing, QR payment parsing, and other utility functions with no UI dependency.

Import via `package:core/core.dart` (re-exported) or directly via `package:fl_utils/fl_utils.dart`.

See [`lib/src/`](lib/src/) for the full list of helpers.
