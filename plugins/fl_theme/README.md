# fl_theme

Design-system/theming plugin for this Flutter base template. Builds `ThemeData` from an `AppDesignSystem` or portable `ThemeJsonConfig`, and registers semantic extensions (`context.themeColor`, `context.textTheme`, `context.screenTheme`, `context.decorationTheme`, `ThemeButton`).

Import via `package:core/core.dart` (re-exported) or directly via `package:fl_theme/fl_theme.dart`.

See [`.agents/skills/fl-theme-usage/SKILL.md`](../../.agents/skills/fl-theme-usage/SKILL.md) for usage, and [`example/`](example/) for the interactive theme playground (JSON import/export, presets, live preview).
